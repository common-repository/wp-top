== Description ==

Adds a button to your articles to offer and vote on Yabama.com.
Including articles's url and the thumbnail of article can be submitted at the same time.

== Installation ==

1. Extract the `wp-vote.php` to your local system prior to upload.
1. Upload `wp-vote.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugin Panel' in WordPress
1. While in the WordPress admin panel, simply Adjustment Placement of the vote button by clicking on the tab for "Options Vote".
